#  Moda Avenue Spring Campaign – Dashboard

##  Project Overview
This project presents a Power BI dashboard analyzing Moda Avenue’s **Spring Campaign**.  
The analysis focuses on **Instagram flash sales** and **influencer collaborations** to evaluate their impact on sales and engagement.

##  Files Included
- **Moda Avenue Style Surge.xlsx** → Raw dataset used for analysis  
- **Moda Avenue Spring Campaign – dashboard.pbix** → Power BI dashboard file  
- **Moda Avenue – dashboard.png** → Dashboard preview image  
- **Moda_Avenue_Style_Surge_Report_Professional.pdf** → Final analysis report with insights & recommendations  

##  Key Insights
- Instagram flash sales drove **60% of total conversions**  
- Influencer campaigns achieved the **highest ROI** during weekends  
- Evening posts (6–9 PM) recorded **top engagement rates**  
- Accessories & Dresses were the **best-selling product categories**

##  Recommendations
- Focus marketing efforts on evening time slots  
- Increase micro-influencer partnerships  
- Continue leveraging Instagram stories for flash sales
